Advanced Data Visualization in R

Complex Data Visualization for Research Findings

Project Overview

Creating sophisticated visualizations to represent complex relationships between internet usage patterns, well-being indicators, and demographic variables.

Visualization Process

1. Data Preparation


# Load required libraries
library(ggplot2)
library(dplyr)
library(tidyr)
library(scales)

# Prepare data for visualization
data_prep <- df %>%
  group_by(education_level) %>%
  summarise(
    avg_importance = mean(internet_importance, na.rm = TRUE),
    avg_wellbeing = mean(wellbeing_index, na.rm = TRUE),
    avg_social = mean(social_connections, na.rm = TRUE)
  )

2. Complex Visualizations


# Create advanced plot
ggplot(data_prep, aes(x = avg_importance, y = avg_wellbeing)) +
  geom_point(aes(size = avg_social, color = education_level), alpha = 0.7) +
  geom_smooth(method = "lm", se = TRUE, color = "darkblue") +
  facet_wrap(~education_level) +
  scale_size_continuous(range = c(3, 10)) +
  theme_minimal() +
  labs(
    title = "Internet Importance vs Well-being by Education Level",
    x = "Average Internet Importance",
    y = "Average Well-being Score",
    size = "Social Connection Level"
  ) +
  theme(
    plot.title = element_text(size = 16, face = "bold"),
    axis.title = element_text(size = 12),
    legend.position = "right"
  )

3. Interactive Elements


# Create interactive visualization using plotly
library(plotly)

p <- ggplot(data_prep, aes(x = avg_importance, y = avg_wellbeing,
                          text = paste("Education:", education_level,
                                     "\nSocial Score:", round(avg_social, 2)))) +
  geom_point(aes(color = education_level)) +
  geom_smooth(method = "lm", se = FALSE) +
  theme_minimal()

ggplotly(p, tooltip = "text")

Key Features

  • Complex multi-variable visualizations
  • Interactive plots for better data exploration
